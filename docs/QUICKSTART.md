# Quickstart

## 1. Clone and install
```bash
git clone <your-repo-url>
cd Code-Kit-Ultra
cp .env.example .env
npm install
```

## 2. Bootstrap
```bash
npm run bootstrap
```

## 3. Preflight
```bash
npm run preflight
```

## 4. First safe scenario
```bash
npm run ck -- init "Build a planning engine for solar CRM" --dry-run
```

## 5. Explore metrics
```bash
npm run ck -- metrics
```
