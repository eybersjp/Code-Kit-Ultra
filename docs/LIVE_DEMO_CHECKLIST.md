## 🕒 T-Minus 10 Minutes: Pre-Flight
- [ ] Run `npm run doctor` to ensure environment health.
- [ ] Verify `ANTIGRAVITY_API_KEY` is set (or prepare to explain the stub fallback).
- [ ] Clear `.codekit/runs/` if you want a fresh "First Success" story.
- [ ] Open the repository in a terminal and a browser (GitHub repo).

## 🎬 Phase 1: The "First Success" (2 Mins)
1.  **Doctor**: Show how we validate health (`npm run doctor`).
2.  **Demo**: Run the main showcase (`npm run demo`).

3.  **Governance**: Show the `gates.json` or the mandatory review policy.
