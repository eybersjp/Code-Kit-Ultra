# First Success Checklist

Use this to confirm your first successful Code Kit Ultra run.

- [ ] Repository cloned
- [ ] `.env.example` copied to `.env`
- [ ] Dependencies installed
- [ ] Bootstrap completed
- [ ] `scripts/doctor.sh` passes
- [ ] `scripts/demo.sh` completes
- [ ] A timestamped folder appears in `artifacts/test-runs/`
- [ ] `run-report.json` exists
- [ ] `report.md` exists
- [ ] You can explain which adapter path was used
- [ ] You can run a second dry-run without confusion

When all boxes are checked, onboarding is successful.
