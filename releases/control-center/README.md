# Release Control Center: Output Index

This directory contains the formal governance records for all Code Kit Ultra releases.

## 📁 Directory Structure
- **manifests/**: Machine-readable JSON files containing the full release state.
- **reports/**: Human-readable Markdown reports and executive summaries.

## 🛠️ Usage
These reports are generated by the `release:control-center` script and are used to provide an auditable trace of every release decision. They are also uploaded as artifacts in the GitHub Actions workflows.
