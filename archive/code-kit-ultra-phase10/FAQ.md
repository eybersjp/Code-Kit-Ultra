# FAQ

## What is Code Kit Ultra?
A governed AI operating system for orchestrating planning, skills, implementation, and promotion workflows.

## Is it safe to try locally?
Yes. Use dry-run mode and the local-safe profile for safer experimentation.

## Does it require Antigravity?
No, but the default public release is Antigravity-first for planning and skills.

## Is it production-ready?
The public release is designed to be ship-safe and observable. Some external adapters may still use placeholder-backed execution paths depending on your setup.

## How do I contribute?
Start with `CONTRIBUTING.md` and use the issue / PR templates.
