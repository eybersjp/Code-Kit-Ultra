# Code Kit Ultra — Mode-Aware Action Runner Pack

This pack adds the next critical execution layer for Code Kit Ultra:
real tool-executing builder actions with guarded execution.

## Included
- action type contracts
- action policy / risk scoring
- mode-aware action runner
- execute command handler
- artifact + log writer
- dry-run support
- rollback metadata stubs
- example builder output contract
- integration notes

## New Runtime Folders
- `.ck/logs/`
- `.ck/artifacts/{runId}/`
- `.ck/runs/`

## Core Goal
Turn the build phase from "AI describes work" into "AI performs work safely".
