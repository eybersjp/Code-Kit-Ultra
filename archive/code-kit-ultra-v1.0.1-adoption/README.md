# Code Kit Ultra

Code Kit Ultra is a governed, observable, specialty-aware AI operating system for structured execution across planning, skills, implementation, automation, and promotion.

## v1.0.1 Adoption Sprint
This package focuses on:
- faster first-run success
- clearer demos
- better troubleshooting
- stronger example-driven onboarding
- better feedback capture

## Who this is for
- developers exploring agentic systems
- teams building governed AI workflows
- operators who need auditability, rollback, and safe dry-runs

## Who this is not for
- users looking for a zero-config hosted SaaS
- teams that do not need governance, routing, or execution visibility

## Install in 10 minutes
```bash
cp .env.example .env
npm install
npm run bootstrap
bash scripts/doctor.sh
bash scripts/demo.sh
```

## Fastest path to first success
1. Read `docs/FIRST_SUCCESS_CHECKLIST.md`
2. Run `bash scripts/doctor.sh`
3. Run `bash scripts/demo.sh`
4. Open the latest folder in `artifacts/test-runs/`
