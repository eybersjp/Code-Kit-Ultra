# Code Kit Ultra — Vertical Slice

This is the first runnable vertical slice for Code Kit Ultra.

It supports:
- CLI intake
- mode-aware clarifying questions
- initial plan generation
- skill selection from a file registry
- gate evaluation
- run report persistence to `.codekit/runs`
- smoke testing

## Quick start

```bash
npm install
npm run typecheck
npm run test:smoke
npm run ck -- init "Build a CRM for solar installers"
```
