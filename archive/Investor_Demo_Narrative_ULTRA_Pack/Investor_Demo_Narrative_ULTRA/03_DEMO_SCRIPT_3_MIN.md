# 3-Minute Demo Script

## 0:00 - Hook
"Most AI tools help write code. Ours helps safely run the engineering system."

## 0:20 - Start with an idea
Show CLI or UI kickoff:
- create run
- interpret objective
- show scoped org/project context

## 0:50 - Show governance
Display:
- risk assessment
- gate awaiting approval
- actor identity
- audit metadata

## 1:30 - Show execution
Resume the run.
Display:
- execution steps
- verification result
- artifacts/logs
- timeline updates

## 2:15 - Show control plane
Move to web or VS Code:
- pending gates
- run list
- audit trail
- tenant-aware status

## 2:45 - Close
"This is the layer between AI generation and trustworthy software delivery."
