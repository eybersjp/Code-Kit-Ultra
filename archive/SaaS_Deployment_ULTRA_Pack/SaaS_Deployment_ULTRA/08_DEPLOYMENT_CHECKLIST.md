# Deployment Checklist

## Before staging
- [ ] secrets populated
- [ ] migrations reviewed
- [ ] smoke tests updated
- [ ] dashboards configured

## Before production
- [ ] release candidate tagged
- [ ] staging passed
- [ ] rollback plan reviewed
- [ ] alerts verified
- [ ] support channels ready

## After production
- [ ] /v1/session healthy
- [ ] run creation healthy
- [ ] audit writes healthy
- [ ] artifact uploads healthy
- [ ] realtime healthy
