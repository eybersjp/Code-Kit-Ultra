# Troubleshooting

## Doctor script fails
Fix the first failing check before moving on.

## Demo script runs but no artifacts appear
Confirm `artifacts/test-runs/` exists and that the shell has write access.

## I don’t understand which adapter was used
Start with the demo and read the routing explanation in the report output.

## I want a safer experience
Stay in local-safe or dry-run style flows until basic onboarding is complete.
